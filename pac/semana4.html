Componentes Utilizados

PLACA ARDUINO UNO R3
Trata-se de uma placa microcontroladora baseada no chip ATmega328, que possui 14 pino digitais I/O (dos quais 6 pode ser usados com saída PWM), 6 entradas analógicas, um cristal oscilador de 16 MHz, uma conexão USB, conector de energia, barramento ICSP e um botão de reset.
CI L293D
Configura-se como um "CI" (circuito integrado) que possibilita, em seus 4 canais de controle, a manipulação total do sentido e a rotação dos motores a ele conectados (4 motores DC, 2 motores de passo ou 2 servo motores). Internamente ele possui 4 chaves e uma fonte de alimentação.
MOTOR CC
Como o nome já denota, trata-se de um motor de corrente contínua de tensão variável (algo em torno de 3V a 9V) cujo consumo de corrente não ultrapasse 1A. Sua finalidade é a conversão de energia elétrica em energia mecânica, gerando uma rotação no seu eixo.
FONTE DE TENSÃO VARIÁVEL
Também denominada de fonte regulável, este equipamento eletrônico de bancada nada mais é que um dispositivo que viabiliza um ajuste fino de tensão e corrente de saída para energizar um circuito eletrônico ou realizar testes de funcionalidade.
FOTORESISTOR
Também conhecido como LDR "Light Dependent Resistor", um fotoresistor é um componente eletrônico, sensível a luminosidade, que tem como objetivo limitar a corrente elétrica que passa sobre ele, tal como um resistor comum, porém dessa vez o grande diferencial é que ele é um resistor variável que tem a sua resistência alterada de acordo com a luz que incide sobre ele.

Programação

//Motor 1:
const int motorPin1  = 5;  
const int motorPin2  = 6;  
//Motor 2:
const int motorPin3  = 10;
const int motorPin4  = 9;  

void setup()
{
    pinMode(motorPin1, OUTPUT);
    pinMode(motorPin2, OUTPUT);
    pinMode(motorPin3, OUTPUT);
    pinMode(motorPin4, OUTPUT);
    pinMode(A3, INPUT_PULLUP);
    pinMode(A4, INPUT_PULLUP);
    pinMode(A5, INPUT_PULLUP); 
    }
void loop()
{
    if (digitalRead(A4) == HIGH){
        digitalWrite(5, HIGH);
        digitalWrite(6, LOW);
        digitalWrite(10, HIGH);
        digitalWrite(9, LOW);    
    } 
    else{
        digitalWrite(5, LOW);
        digitalWrite(6, HIGH);
        digitalWrite(10, LOW);
        digitalWrite(9, HIGH);
    }

    if (digitalRead(A3) == LOW){
        digitalWrite(5, HIGH);
        digitalWrite(6, LOW);
        digitalWrite(10, LOW);
        digitalWrite(9, HIGH);
        delay(2100);
    } 

    if (digitalRead(A5) == LOW){
        digitalWrite(5, LOW);
        digitalWrite(6, HIGH);
        digitalWrite(10, HIGH);
        digitalWrite(9, LOW);
        delay(2100);
    } 
}
